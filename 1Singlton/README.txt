Задание 1

1. Реализовать программу по паттерну Singleton в ленивой реализации.
2. В отдельных файлах создать три класса Cat, Dog, Rat.
3. Создать интерфейс Animal в классах Cat, Dog, Rat.
4. Создать простой GUI с текстовым окном и кнопкой 
5. В статическом блоке класса Solution вызвать метод readInitedAnimal
6. В методе readInitedAnimal предусмотреть ввод одного параметра типа String.
В окне вывода результата должно появиться имя одной из констант интерфейса Animal

Реализовано и содержит в себе
1. папка animals 
    Интерфейс Animal: служит для хранения констант, которые определяют имена определённых животных. Это общий интерфейс для всех животных
    Классы:
        Cat,Dog,Rat реализуют интерфейс Animal и возвращают имя животного через метод getName(), определённый в интерфейсе Animal
2.Класс Solution: содержит логику для демонстрации использования интерфейса Animal и его реализаций
3. Класс SimpleGUI: предоставляет графический интерфейс для взаимодействия с пользователем

Итог программы включает в себя графический интерфейс в который можно ввеси назнание животного и получить его имя есть такое животное существует, иначе вывод будет "Unknown Animal"
