Это приложение представляет собой простой графический интерфейс пользователя (GUI), который демонстрирует использование паттерна "Абстрактная фабрика" (Abstract Factory) для отображения цен на автомобили из разных стран. Пользователи могут выбрать страну и модель автомобиля, чтобы увидеть его стоимость.

Как это работает
Приложение включает в себя два основных компонента:

Логику фабрик и моделей автомобилей (Cars.java): Определяет интерфейсы и классы для моделей автомобилей (Lada, Ferrari, Porshe) и фабрик по созданию автомобилей в зависимости от страны (Беларусь, Россия). Каждая модель автомобиля имеет свою цену, которая отличается в зависимости от страны.

Графический интерфейс пользователя (CarsPriceGUI.java): Предоставляет GUI, через который пользователи могут выбрать страну и модель автомобиля из выпадающих списков и получить информацию о стоимости выбранного автомобиля.

Запуск приложения
Для запуска приложения необходимо скомпилировать оба файла (Cars.java и CarsPriceGUI.java) и запустить CarsPriceGUI.

Откройте терминал или командную строку.
Перейдите в каталог проекта, где находятся файлы исходного кода.

Скомпилируйте файлы, используя команду:
javac Cars.java CarsPriceGUI.java

Запустите приложение командой:
java CarsPriceGUI

Интерфейс и использование
После запуска приложения перед вами откроется окно с двумя выпадающими списками (для выбора страны и модели автомобиля) и кнопкой "Получить цену". Выберите страну и модель автомобиля, затем нажмите на кнопку, чтобы увидеть стоимость выбранного автомобиля. Цена будет отображена ниже кнопки.
