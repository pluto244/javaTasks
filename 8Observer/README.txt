Эта программа представляет собой чат-приложение с графическим интерфейсом, разработанное на языке программирования Java с использованием библиотеки Swing. Приложение реализует паттерн "Наблюдатель" (Observer) для управления подпиской пользователей на обновления чата. Оно поддерживает основные функции чата, такие как добавление и удаление пользователей, отправка сообщений всем пользователям и адресная отправка сообщений между конкретными пользователями. Программа также включает удобный графический интерфейс, который позволяет пользователям легко взаимодействовать с приложением.

Интерфейс Observer определяет метод update(String message), который вызывается для обновления наблюдателя при получении нового сообщения.

Интерфейс Subject определяет методы для добавления и удаления наблюдателей, уведомления их об изменениях и отправки адресных сообщений.

Класс Chat реализует интерфейс Subject и управляет списком наблюдателей, а также оповещает их при получении нового сообщения.

Класс User реализует интерфейс Observer и представляет собой конкретного пользователя чата. Пользователь может присоединяться к чату, покидать его, отправлять сообщения всем пользователям и отправлять личные сообщения конкретным пользователям.

Класс ChatGUI представляет графический интерфейс приложения. Он предоставляет функции для добавления и удаления пользователей, отправки сообщений и выбора текущего пользователя.

Класс ChatDemo содержит метод main для запуска приложения с графическим интерфейсом.

Функциональность
Добавление пользователя: Пользователь может ввести свое имя в текстовое поле и нажать кнопку "Add User", чтобы присоединиться к чату. При добавлении пользователя отправляется оповещение всем пользователям о присоединении нового участника.
Удаление пользователя: Пользователь может нажать кнопку "Remove User", чтобы покинуть чат. При удалении пользователя отправляется оповещение всем пользователям о выходе участника.
Отправка сообщений всем пользователям: Пользователь может ввести сообщение в текстовое поле и нажать кнопку "Send", чтобы отправить сообщение всем пользователям чата.
Отправка личных сообщений: Пользователь может выбрать другого пользователя из выпадающего списка и ввести сообщение, затем нажать кнопку "Send Direct Message", чтобы отправить личное сообщение выбранному пользователю.
Выбор текущего пользователя: Пользователь может выбрать другого пользователя из списка пользователей и нажать кнопку "Choose User", чтобы видеть все сообщения, относящиеся к выбранному пользователю, включая общие и личные сообщения.


Список пользователей: В левой части интерфейса отображается список всех пользователей чата.
Список сообщений: В центре интерфейса отображается список сообщений текущего пользователя.
Текстовое поле для ввода сообщений: В нижней части интерфейса находится текстовое поле для ввода сообщений и кнопка для отправки сообщений.
Управление пользователями: Справа от списка сообщений находятся текстовое поле для ввода имени пользователя и кнопки для добавления и удаления пользователей.
Отправка личных сообщений: В правой части интерфейса также находится выпадающий список для выбора пользователя и кнопка для отправки личных сообщений.
