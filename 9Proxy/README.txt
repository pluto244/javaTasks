КОД 1234


Программа "Контроль доступа к двери" разработана для управления доступом к двери с использованием паттерна проектирования Proxy (Заместитель). Она представляет собой графическое приложение на языке Java, которое позволяет пользователям открывать и закрывать дверь, вводя правильный код или используя эмуляцию карты. Программа также включает визуальную анимацию, показывающую процесс открытия и закрытия двери.

Интерфейс Door определяет базовые методы для управления дверью: open(), close(), и isOpen(). Этот интерфейс реализуется классами RealDoor и ProxyDoor.

Класс RealDoor представляет реальную дверь. Он содержит методы для открытия и закрытия двери, а также для проверки её состояния. При выполнении этих действий пользователю выводятся уведомления с помощью JOptionPane.

Класс ProxyDoor реализует интерфейс Door и выступает в роли заместителя для RealDoor. Он добавляет проверку учетных данных (кода или карты) перед выполнением операций с дверью. Если введенный код или карта верны, дверь открывается или закрывается. Также включены методы для очистки учетных данных после каждой операции.

Класс DoorAccessControl представляет собой графический интерфейс пользователя для управления доступом к двери. Он использует Swing-компоненты для создания окна с полем ввода кода, кнопками для открытия и закрытия двери, а также кнопкой для эмуляции карты. Программа включает анимацию процесса открытия и закрытия двери.

Анимация открытия и закрытия двери реализована с помощью Timer и JLabel. Изображения дверей (закрытая, открывающаяся, открытая) хранятся в папке /images и загружаются в массивы openImages и closeImages. Когда дверь открывается или закрывается, соответствующая анимация отображается на экране.
